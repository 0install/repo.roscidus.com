This Zero Install repository, still under construction and experimental, provides feeds for programs where the upstream authors do not provide feeds of their own. The repository is kindly hosted by the FSF, and hosts only Free software.

The scripts used to build this repository are maintained in a GIT repository. See 0install.net for more information about Zero Install.

Currently available sections:

  • The E language (secure distributed programming language)
  • Java (OpenJDK 6)
