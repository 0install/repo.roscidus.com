This 0install repository provides feeds for programs and libraries where the upstream authors do not provide feeds of their own.
It also serves as a semi-official repository for basic infrastructure components such as libraries and runtime environments.

The contents of this repository are maintained in a GIT repository. Some packages are created by hand, others are generated automatically from Debian packages.
This website is kindly hosted by the FSF, and hosts only Free software.

See 0install.net for more information about Zero Install.
