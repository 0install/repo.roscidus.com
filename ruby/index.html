repo.roscidus.com: Ruby

Ruby is a dynamic, open source programming language with a focus on simplicity and productivity.

To add irb (Interactive Ruby) to your PATH:

      $ 0alias irb http://repo.roscidus.com/ruby/ruby irb

To add gem (package management framework for Ruby) to your PATH:

      $ 0alias gem http://repo.roscidus.com/ruby/ruby gem

To specify Ruby as the runner in a feed:

      <command name="run" path="myapp.rb">
        <runner interface="http://repo.roscidus.com/ruby/ruby"/>
      </command>
