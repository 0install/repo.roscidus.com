repo.roscidus.com: Git

Git is a fast, scalable, distributed revision control system.

To add git to your PATH:

      $ 0alias git http://repo.roscidus.com/git/core

To integrate Git and related tools in your Windows desktop environment:

      $ 0install integrate http://repo.roscidus.com/git/windows

To specify a dependency on Git in your own feed:

      <requires interface="http://repo.roscidus.com/git/core">
        <executable-in-path name="git"/>
      </requires>

Available feeds

  • Git Core (the main git command on any platform)
  • Git for Windows (git and related tools like bash on Windows)
