repo.roscidus.com: Go Programming Language

Go is an open source programming language that makes it easy to build simple, reliable, and efficient software.

To add Go-related commands to your PATH:

      $ 0alias go http://repo.roscidus.com/golang/go
      $ 0alias godoc http://repo.roscidus.com/golang/go godoc
      $ 0alias gofmt http://repo.roscidus.com/golang/go gofmt
